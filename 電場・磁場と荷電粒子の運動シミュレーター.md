# 電場・磁場と荷電粒子の運動 シミュレーター AI 指示書

## 0. 共通事項

本仕様書は、既存の参考プロジェクト群と同じ設計思想・運用前提で動作する Web 教材の作成指示書である。個別機能に入る前に、全テーマ共通の前提を以下にまとめる。以降の章は、この共通事項を前提として本テーマ固有の要件のみを記述する。

### 0.1 参考プロジェクト

- [GEO-LEARNER](https://takayuki-minagawa.github.io/GEO-LEARNER/) — 観測視点切替 + 学習モジュールの三層構成
- [weather-front-sim](https://takayuki-minagawa.github.io/weather-front-sim/) — 時系列操作 + 観測値グラフ + 学習パネル + クイズ
- [web-planetarium](https://takayuki-minagawa.github.io/web-planetarium/) — 地点・時刻に応じた星空描画と i18n / テーマ切替

これらと同じ設計思想（操作パネル＋可視化ビュー＋学習パネル＋クイズ、i18n、ライト／ダーク切替、静的配信）を踏襲する。

### 0.2 共通仕様

- 静的フロントエンド構成とする（React + TypeScript + Vite を基本、軽量なら Vanilla JS + HTML + CSS も可）
- GitHub Actions により GitHub Pages へデプロイする
- 日本語 UI を基本とし、英語切替（i18n）に対応する
- ライト／ダークテーマ切替に対応する
- PC・タブレット・スマホ対応のレスポンシブ UI にする
- 画面は「操作パネル」「可視化ビュー」「学習パネル（解説＋クイズ）」の三層構成を基本とする
- 教育用途のコンテンツであり、実測・予報・診断など厳密な数値提供を目的にしない
- 公開リポジトリ前提の表現で README と仕様書を書き、個人情報・認証情報・秘密鍵は含めない

### 0.3 共通で再利用したいコンポーネント

- タイムスライダー、再生・一時停止・速度変更コントロール
- 学習パネル・クイズフレーム（状態に応じた解説文生成と正誤判定）
- i18n ロード機構、ライト／ダークテーマ切替
- 3D シーン基盤（Three.js + React Three Fiber）: 3D 系テーマで共用
- 2D グラフ・波形描画基盤（Canvas / D3.js）: 時系列グラフ系テーマで共用
- SVG 編集系 UI（図形選択・ドラッグ・履歴）: 図形操作系テーマで共用

### 0.4 学習パネルの共通方針

- 現在の状態・入力に対する解説を自動生成する
- 観察の着目点や誘導テキストを出す
- クイズモードを併設し、主要な誤解ポイントを問題化する
- 言語切替に追従して文言が再生成される構造にする

## 1. 目的

一様電場・一様磁場・組み合わせ場の中を運動する荷電粒子の軌跡をリアルタイム描画し、ローレンツ力や円運動・らせん運動・サイクロイド運動を体感できる Web 教材を作成する。

高校物理「電磁気」分野の定番であり、3 次元の運動をイメージすることが難しい単元を、3D ビューと軌跡描画で直感的に理解できることを目指す。

- 電場中の等加速度運動
- 磁場中の等速円運動（ローレンツ力）
- 電場＋磁場のサイクロイド運動・ドリフト
- サイクロトロン運動の周期 T = 2πm/(qB)
- 速度選別器（E × B ドリフト）

## 2. 開発方針

### 2.1 基本方針

- 静的フロントエンド Web アプリ
- GitHub Pages 公開
- PC／スマホ対応、日英切替、ライト／ダーク

### 2.2 推奨技術スタック

- React + TypeScript + Vite
- 3D 描画: Three.js + React Three Fiber + drei
- スタイリング: Tailwind CSS または CSS Modules

### 2.3 画面構成

- **3D シミュレーションビュー**: 一様電場・磁場の矢印グリッド、粒子、軌跡
- **パラメータパネル**: 電場・磁場ベクトル、粒子の質量、電荷、初期位置、初速度
- **数値・グラフパネル**: 速度、エネルギー、角周波数、周期、軌跡の時間変化
- **学習パネル**: 解説、クイズ

---

## 3. 必須機能

### 3.1 場の設定

- 一様電場 E = (Ex, Ey, Ez) を各成分スライダーで調整
- 一様磁場 B = (Bx, By, Bz) を各成分スライダーで調整
- プリセット: 電場のみ、磁場のみ、平行、垂直、速度選別器
- 場を 3D 空間内に矢印グリッドで表示（表示密度は調整可能）

### 3.2 粒子の設定

- 質量 m、電荷 q（正／負、電子・陽子プリセット）
- 初期位置、初速度ベクトル
- 粒子を複数配置できる（比較運動用）

### 3.3 運動シミュレーション

- 運動方程式: m dv/dt = q(E + v × B)
- 数値積分: 4 次の Runge–Kutta または Boris 法（磁場運動で安定）
- 再生・一時停止・リセット・倍速
- 軌跡を色付きラインで残像表示
- サイクロトロン運動では円の直径、半径、周期を図示

### 3.4 可視化

- 粒子の速度ベクトルと力ベクトルをリアルタイム表示
- 電場ベクトル・磁場ベクトルを静的に表示
- 軌跡の 3D ラインに時間色グラデーション
- 任意視点へのカメラ回転、ズーム、リセット

### 3.5 数値・グラフ

- 運動エネルギー、位置、速度、加速度、周期の時系列プロット
- 磁場のみの場合の半径 r = mv/(qB) をリアルタイム表示
- サイクロトロン周期のリアルタイム値

### 3.6 学習パネル

- 現在の状態に対する解説
- クイズ
  - ローレンツ力の向きクイズ（フレミング左手の法則）
  - サイクロトロン半径の計算
  - 速度選別器の条件判定

## 4. あると良い機能

- 粒子加速器のプリセット（サイクロトロン、ベータトロン、直線加速器）
- 荷電粒子ビーム（多粒子）モード
- 磁場不均一の拡張（ミラー磁場・双極磁場）
- URL クエリで条件共有
- PWA 対応

---

## 5. データ・計算設計

### 5.1 数値積分

- Boris 法: 磁場中の回転運動を長時間保っても発散しにくい
- 時間刻み dt を自動調整（速度に応じて縮小）

### 5.2 表示スケール

- 粒子サイズと矢印サイズは視認優先のスケール
- 実スケール表示モードを切替可能

### 5.3 プリセット

`fieldPresets.ts` に代表例を定義。

---

## 6. アニメーションとビジュアル

- 粒子の軌跡は時間色付きの 3D ライン
- 速度ベクトル・力ベクトルは色分け（速度: 青、力: 赤）
- 場のベクトル矢印は半透明で邪魔にならない密度
- カメラを自動追尾するオプションあり
