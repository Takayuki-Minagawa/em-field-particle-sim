# 電場・磁場と荷電粒子の運動シミュレーター

電場・磁場中を運動する荷電粒子の軌跡を、ブラウザ上で 3D 表示しながら学べる教育用 Web アプリケーションです。  
高校物理の電磁気分野を主な対象とし、ローレンツ力、等速円運動、らせん運動、電場と磁場の組み合わせによるドリフトや速度選別器の考え方を、視覚的に確認できる構成にしています。

このリポジトリは静的フロントエンドのみで構成されており、GitHub Pages での公開を前提としています。

## 公開リポジトリとしての前提

- 本アプリは教育目的の教材です。研究用、実験制御用、設計用の厳密な数値計算ソフトウェアではありません。
- 画面上の値は理解しやすさを優先した正規化表現であり、実験室スケールの実測値を直接再現するものではありません。
- アプリケーション専用のサーバー、認証機能、ユーザー情報保存機能は含みません。
- リポジトリ内に個人情報、秘密鍵、認証情報を含めない前提で管理してください。
- 本アプリは外部バックエンドを必要としませんが、表示フォントの取得で外部配信資産を参照する場合があります。

## できること

- 一様電場 `E = (Ex, Ey, Ez)` の各成分を調整
- 一様磁場 `B = (Bx, By, Bz)` の各成分を調整
- 電場のみ、磁場のみ、平行場、垂直場、速度選別器のプリセット切替
- 粒子の質量、電荷、初期位置、初速度の調整
- 複数粒子を同時に配置して軌道を比較
- 3D ビューで粒子、軌跡、速度ベクトル、力ベクトル、場の矢印を表示
- 再生、一時停止、リセット、再生速度変更、軌跡長の変更
- 速度、運動エネルギー、サイクロトロン半径、位置量の時系列表示
- 状態に応じた説明文の生成
- クイズ形式での確認問題
- 日本語 / 英語切替
- ライト / ダークテーマ切替
- PC、タブレット、スマートフォン向けのレスポンシブ表示

## 画面構成

本アプリは、以下の 3 層構成を基本としています。

### 1. 操作パネル

- 場のプリセット選択
- 電場・磁場ベクトルの調整
- 粒子の追加、削除、選択
- 質量、電荷、初期位置、初速度の調整
- 再生制御、軌跡長、カメラ追尾、視点リセット

### 2. 3D シミュレーションビュー

- 電場、磁場の矢印グリッド表示
- 粒子の現在位置表示
- 軌跡ラインの表示
- 速度ベクトル、ローレンツ力ベクトルの表示
- 任意視点からの回転、ズーム、追尾
- 条件に応じたサイクロトロン半径のガイド表示

### 3. 学習パネル / 数値パネル

- 現在状態に応じた説明文
- 力の向き、半径、速度選別器に関するクイズ
- 速度、エネルギー、半径、位置量の簡易グラフ

## プログラム仕様

### 計算モデル

荷電粒子の運動は、以下のローレンツ力に基づいて更新します。

`m dv/dt = q(E + v × B)`

- `m`: 質量
- `q`: 電荷
- `E`: 電場ベクトル
- `B`: 磁場ベクトル
- `v`: 粒子速度

### 数値積分

- 数値積分には `Boris 法` を採用しています。
- 磁場中の回転運動を比較的安定して表現できるため、教育用のリアルタイム表示に適しています。
- 内部では粒子速度に応じて時間刻みを調整し、極端な発散を抑えるようにしています。

### 物理量表示

アプリ内では、以下の値を補助的に表示します。

- 速度
- 運動エネルギー
- ローレンツ力の大きさ
- 加速度の大きさ
- サイクロトロン半径
- サイクロトロン周期

サイクロトロン半径と周期は、磁場または電荷が 0 に近い場合には定義しません。

### 可視化仕様

- 電場ベクトルは青系
- 磁場ベクトルは暖色系
- 速度ベクトルは青
- 力ベクトルは赤
- 粒子軌跡は粒子ごとに色分け
- 軌跡は時間経過に応じた色の変化を持つラインとして表示

## 学習コンテンツ仕様

学習パネルでは、現在の場と粒子状態から以下を生成します。

- どの場が支配的か
- 力が主に速度の向きを変えているのか、速さを変えているのか
- 円運動、らせん運動、ドリフトの見え方
- `E × B` ドリフトの考え方

クイズは以下の 3 系統を実装しています。

- ローレンツ力の向き
- サイクロトロン半径の見積もり
- 速度選別器の通過条件 `v = E / B`

## 主要ファイルの役割

- `src/App.tsx`

  パラメータ操作 UI を担当します。

- `src/data/translations.ts`
  日本語 / 英語の表示文言を管理します。

## GitHub Pages 公開

GitHub Pages 用の Actions ワークフローを同梱しています。

- 定義ファイル: `.github/workflows/deploy.yml`
- 対象ブランチ: `main`
- 主な処理:
  - 依存関係インストール
  - テスト実行
  - ビルド実行
  - `dist/` を GitHub Pages へデプロイ

`vite.config.ts` では、GitHub Actions 実行時にリポジトリ名をもとに `base` を設定するため、リポジトリ配下の Pages 公開にも対応します。

## テスト方針

現在は主に軽量なロジックテストを実装しています。

- ベクトル演算の基本挙動
- 磁場のみの場合のエネルギー保存傾向
- 電場中での加速挙動
- 学習パネル用の説明文とクイズ生成

UI の視覚確認や操作性確認は、必要に応じてブラウザ上で追加確認してください。

## 制約と注意

- 教材用途のため、物理定数や実単位系を厳密に再現する設計ではありません。
- 多数粒子や高密度描画を前提とした最適化は行っていません。
- 3D 描画ライブラリの都合により、ビルド時に比較的大きな JavaScript チャンクが生成されることがあります。
- ブラウザ性能や端末性能によって描画の滑らかさは変わります。
