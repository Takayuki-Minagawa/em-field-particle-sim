# 電場・磁場と荷電粒子の運動シミュレーター

高校物理の電磁気分野を対象にした、静的配信向けの 3D Web 教材です。  
一様電場・一様磁場・組み合わせ場における荷電粒子の運動をリアルタイムで描画し、ローレンツ力、等速円運動、らせん運動、E x B ドリフト、速度選別器の条件を比較しながら学べます。

## 主な機能

- React + TypeScript + Vite による静的フロントエンド
- Three.js / React Three Fiber による 3D シーンと軌跡描画
- Boris 法による荷電粒子の時間発展
- 電場・磁場ベクトル、粒子の質量・電荷・初期条件、複数粒子比較
- 電場のみ、磁場のみ、平行場、垂直場、速度選別器のプリセット
- 速度、エネルギー、半径、位置の時系列グラフ
- 状態説明の自動生成と 3 問のクイズ
- 日本語 / 英語切替、ライト / ダークテーマ、レスポンシブ対応
- GitHub Actions による GitHub Pages デプロイ

## 開発

```bash
npm install
npm run dev
```

## テストとビルド

```bash
npm test
npm run build
```

## デプロイ

`main` ブランチへの push で GitHub Actions が `dist/` を生成し、GitHub Pages へ公開します。
